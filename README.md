## 설치법
1. zip파일로 다운로드
2. 압축해제
3. 크롬실행
4. chrome://extensions/ 접속
5. 개발자모드 on
6. 좌측 상단 "압축해제된 확장 프로그램을 로드합니다." 클릭
7. 압축해제한 폴더에서 GotoTop 선택

## 사용법
- Alt + Q : 확장프로그램 팝업 띄우기
- Alt + A : 위로 이동
- Alt + S : 아래로 이동
- 원래 QWE 로 하려고 했는데 크롬 맵핑때문에 못함
